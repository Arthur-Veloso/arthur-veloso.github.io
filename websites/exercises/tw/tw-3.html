Tailwind CSS Documentation

A utility-first CSS framework packed with classes like flex, pt-4, text-center and rotate-90 that can be composed to build any design, directly in your markup.

What is Tailwind CSS?

Tailwind CSS is a highly customizable, low-level CSS framework that gives you all of the building blocks you need to build bespoke designs without any annoying opinionated styles you have to fight to override.

Unlike traditional CSS frameworks like Bootstrap or Foundation, Tailwind doesn't come with pre-designed components. Instead, it provides utility classes that let you build your own unique designs.

Pro Tip: Use the Tailwind CSS IntelliSense extension in VS Code for autocomplete and syntax highlighting.

Installation

Using CDN

For quick prototyping or simple projects, you can include Tailwind directly from the CDN: 

<script src="https://cdn.tailwindcss.com"></script>

Using npm

For production projects, we recommend installing Tailwind as a PostCSS plugin: 

npm install -D tailwindcss
npx tailwindcss init

Utility-First Fundamentals

Instead of writing custom CSS, you build components by combining utility classes:

Example Button

<button class="bg-blue-500 hover:bg-blue-700 text-white font-bold py-2 px-4 rounded">
    Primary Button
</button>

Benefits of Utility-First

  • You aren't wasting energy inventing class names
  • Your CSS stops growing exponentially
  • Making changes feels much safer
  • You're building a design system, not painting by numbers

Responsive Design

Tailwind uses a mobile-first breakpoint system. By default, styles take effect at all screen sizes, and you can conditionally apply styles at larger sizes using responsive prefixes:

Responsive Grid Example

Column 1
Column 2
Column 3
<div class="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-4">
    <div>Column 1</div>
    <div>Column 2</div>
    <div>Column 3</div>
</div>

Breakpoint Prefixes

Prefix Min Width CSS
sm 640px @media (min-width: 640px)
md 768px @media (min-width: 768px)
lg 1024px @media (min-width: 1024px)
